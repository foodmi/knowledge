# App FoodMi -- Features

## Features existantes

### Scan IA
- [ ] **Photo scan** : prendre ou importer une photo -> identification aliments + calories/macros
- [ ] **Scan frigo** : photographier son frigo -> suggestions de repas
- [ ] **Scan ticket de caisse** : extraire les aliments d'un ticket
- [ ] **Scan barcode** : scanner code-barres produits

### Journal alimentaire
- [ ] Log repas (petit-dej, dejeuner, diner, snack)
- [ ] Historique des repas avec photos
- [ ] Recherche aliments dans la base
- [ ] Ajout manuel (quick add macros)

### Dashboard
- [ ] Anneau calories central avec progression
- [ ] Barre objectif atteint (%)
- [ ] Apercu macros du jour (proteines, glucides, lipides)
- [ ] Liste repas du jour
- [ ] Recompense du jour
- [ ] Defis du moment

### Assistant IA
- [ ] Chat nutritionnel intelligent
- [ ] Suggestions de repas personnalisees
- [ ] Generation de liste de courses
- [ ] Conseils contextualises (mascotte)

### Sante
- [ ] Suivi poids (graphiques)
- [ ] Glycemie
- [ ] Tension arterielle
- [ ] Integration HealthKit (iOS) / Google Fit (Android)
- [ ] Rapports sante IA

### Exercice
- [ ] Log exercices (course, muscu, manuel)
- [ ] Intensite, duree, distance
- [ ] Calories brulees

### Progres
- [ ] Graphiques evolution (poids, calories, macros)

- [ ] Photos progres

### Gamification
- [ ] Systeme XP

### Profil

### Onboarding
